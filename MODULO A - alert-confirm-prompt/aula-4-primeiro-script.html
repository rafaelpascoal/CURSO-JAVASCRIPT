Criando o seu primeiro script

window.alert ('')

Comando em JavaScript que cria mensagem (com a informação que o programador quiser) em uma janela independente.

Podemos, também, utilizar apenas alert (''), sem o window.

  • window.alert ('')
  • alert ('')

Lembrando que o conteúdo de HTML criado para a página não aparecerá enquanto o usuário não selecionar o botão OK na janela de alerta.

window.confirm ('')

Comando em JavaScript que cria mensagem/pergunta (com a informação que o programador quiser) em uma janela independente.

Podemos, também, utilizar apenas confirm (''), sem o window.

  • window.confirm ('')
  • confirm ('')

Lembrando que o conteúdo de HTML criado para a página não aparecerá enquanto o usuário não selecionar o botão OK na janela de alerta.

window.prompt ('')

Comando em JavaScript que cria mensagem/pergunta (com a informação que o programador quiser) em uma janela independente e que exige uma resposta.

Podemos, também, utilizar apenas prompt (''), sem o window.

  • window.prompt ('')
  • prompt ('')

Lembrando que o conteúdo de HTML criado para a página não aparecerá enquanto o usuário não selecionar o botão OK na janela de alerta.

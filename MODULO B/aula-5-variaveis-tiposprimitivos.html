Comandos Básicos do JavaScript - Variáveis e Tipos Primitivos

=

Chamamos o sinal = em JavaScript de RECEBE (não usamos o significado de igual como é na matemática);

VAR

Para facilitar o aprendizado, dizemos que as variáveis (VAR) se assemelham a vagas de estacionamento (espaços em que os carros serão estacionados);

Na linguagem JavaScript, além de utilizarmos a palavra var, também podemos utilizar a palavra let

Essas variáveis podem receber nomes e valores em números e/ou palavras. Exemplos:

  • var n1 = 5 (Significa que o var com nome n1 receberá o valor de 5);
  • var n2 = 8.5 (Significa que o var com nome n2 receberá o valor de 8.5);
  • var n3 = 15 (Significa que o var com nome n3 receberá o valor de 15);
  • var s1 = "JavaScript"
  • var s2 = 'Curso em Vídeo'
  • var s3 = `Guanabara`

No segundo exemplo (em que os valores são referentes a palavras), percebemos que é possível delimitar um string [sequências de caracteres alfanuméricos (letras, números e/ou símbolos)], ou seja, o que será recebido pelas variáveis de três formas: "" (entre aspas duplas), '' (entre aspas simples ou apóstrofos) e `` (entre acentos crase).

O nome de cada variável é conhecido como identificador e esses identificadores devem seguir algumas regras, como:.

  • Podem começar com letra, $ ou _;
  • Não podem começar com números;
  • É possível usar letras ou números;
  • É possivel usar acentos e símbolos;
  • Não podem conter espaços;
  • Não podem ser palavras reservadas (como alert, prompt, confirm, var, etc)

DICAS

  • Maiúsculas e minúsculas fazem diferença (se eu criar uma variável A maiúsculo, devo identificá-las apenas com A maiúsculo e não utilizar o a minúsculo);
  • Tente escolher nomes coerentes para as variáveis (não usar n1, n2, n3 para tudo - utilizar variáveis que facilitem a sua identificação e sua relação com os valores aplicados: var salário para o valor do salário, var idade para o valor idade, e assim por diante);

VARIÁVEIS PELA VISÃO DO JAVASCRIPT

  • NUMBER - 5, 18, -12, 0.5, -15.9, 3.14, 8.0 - Number (Números/variáveis numéricas);
    • Infinity
    • NaN
  • STRING - "Google", 'JavaScript', `Maria` - String (pode ser, também, uma cadeia de números: CPF, telefone, RG - pois possuem pontos, traços, são mais complexos que apenas números. Por isso não estão classificados meramente como numbar);
  • BOOLEAN - True, False (Ligado/Desligado, Sim/Não, ou seja, é o tipo de dado que pode ter somente um de dois valores) - Boolean (Valor Boleano ou Valor Lógico)
